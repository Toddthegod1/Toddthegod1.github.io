## Hi, I'm Todd Klinger

I'm a Computer Science and Data Analytics student at Dickinson College with a minor in Mathematics. I build practical software and data projects involving cloud deployment, APIs, analytics, and interactive web applications.

My work focuses on turning technical ideas into tools people can actually use — whether that means designing an interactive API specification, deploying a full-stack app to the cloud, or building visualizations that make data easier to understand.

### What I Work On
- **Software & web apps:** full-stack and front-end projects with JavaScript, TypeScript, Python, and Java
- **Data analytics:** cleaning, analyzing, and visualizing data using Python, R, SQL, pandas, and matplotlib
- **Cloud & deployment:** AWS services including EC2, RDS, and S3
- **APIs & documentation:** designing and documenting APIs with OpenAPI / Swagger
